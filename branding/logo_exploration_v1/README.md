---

## Evaluation criteria

When selecting a direction for v2 final logo:

1. **16px legibility** (favicon)
2. **512px presence** (Zid app icon)
3. **Misread test** — not sync, cart, chat, or send
4. **Dashboard coexistence** — does not clash with nav icons
5. **RTL mirror** — path flips without semantic loss
6. **Single-color reproduction**
