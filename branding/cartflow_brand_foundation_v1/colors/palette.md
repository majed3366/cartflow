# CartFlow Official Color Palette v1

**Status:** Frozen — approved Visual Identity board  
**Do not** add colors outside this palette without explicit governance approval.

| Token | Hex | Swatch role |
|-------|-----|-------------|
| `cf-primary` | `#1E6B4A` | Logo C arc, headings, primary buttons, admin sidebar |
| `cf-secondary` | `#2DA36A` | “Flow” wordmark, secondary accents |
| `cf-accent` | `#A7E3C4` | Logo F bars, node, highlights |
| `cf-background` | `#E9F7EF` | Page / section backgrounds |
| `cf-surface` | `#F6FBF8` | Cards, modals, inputs |
| `cf-text` | `#344054` | Body text, labels |

## Usage rules

- **One primary green** per view for main CTA; secondary for supporting actions.
- **White** (`#FFFFFF`) allowed for card elevation on top of `cf-surface` / `cf-background`.
- **Semantic status colors** (success/warning/error) may appear in UI data states only — not as brand replacements.
- **WhatsApp green** is never a brand color.

## Files

- `tokens.json` — machine-readable tokens
- `tokens.css` — CSS custom properties for product UI adoption
